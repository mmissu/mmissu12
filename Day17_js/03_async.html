비동기 통신(asynchronous commuication) : AJAX(ajax)

  • 데이터를 받는 방식(통신) : 동기통신, 비동기통신
  • 동기통신(synchronous) : 동시에 일어난다.
    요청을 하면 응답을 바로 받는다는 의미
  • 비동기통신(asynchronous) : 동시에 일어나지 않는다.
    요청을 하면 그 자리에서 결과가 주어지지 않을 수 있다.
  • 동기방식 : 설계가 간단하고 직관적,
    결과가 주어지기전까지는 아무것도 못 하고 대기해야 하는 단점이 있음
  • 비동기방식 : 설계가 복잡함,
    결과가 주어지는데 시간이 걸리더라도 그동안 다른 작업이 가능하여 효율적임
  • async await : Promise 객체를 이용하여 데이터 통신을 편리하게 해주는(제공되는) 키워드
  • async 키워드 : 함수선언 키워드(function)앞에 사용하면 그 함수의 리턴을 프로미스 객체타입으로 만들어준다.
  • await 키워드 : 순차적 코드 실행시 데이터 통신과 같은 지연로딩
    또는 지연 실행 같은 상황이 발생되면 코드의 실행 순서를 순차적으로 진행할 수 있도록 보증한다.
    즉, 이전 라인의 코드 실행이 완료될 때 까지 다음 라인을 기다리도록 하는 기능
