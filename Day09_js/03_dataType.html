변수나 상수에 할당할 수 있는 값의 종류

  • 숫자 : Number => 정수, 실수, Nan(not a number : 숫자가 아님)
  • 문자 : String => " ",' ',` ` 로 감싼 값
  • 불리언 : Boolean = true /false
  • 객체 : OBject => 1개 이상의 데이터가 특정 구조를 가지고 있는 형태
    { 키 : 값 } ({ }) : 아무것도 없는 1개의 객체로 취급 *null로 뜨지 않음
  • 배열 : Array => [1,2,3,....] 형태로 이루어진 데이터 집합
    (순서, 길이가 있음)
  • 알 수 없는 값 : null => 데이터가 존재하나 파악이 불가한 경우
  • 미지정된 값 : undefined => 값이 할당되지 않음
  • 데이터 타입을 확인하는 명령어 : typeof 변수 / 값

변수의 관리

  • 기본자료형 : 값이 메모리의 주소값이 가리키는 곳에 직접 할당
    number, boolean, string(참조자료형이지만 사용편의상 같이 사용)
  • 참조자료형(reference Type) : 값이 저장되어 있는 주소를 할당 string, object , array, function, null, undefined
  • == 같다(값만 비교), != 같지않다 === 같다(값과 타입이 같은지 비교) !-- 같아(값과 타입이 같지 않은지 비교)
